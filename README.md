- **Article X (Link Sovereignty)**: নূরনেক্সাস কোনো থার্ড-পার্টি রিডাইরেক্ট সার্ভিসের ওপর নির্ভর করবে না। সকল লিঙ্ক সরাসরি সার্বভৌম মেশ থেকে পরিচালিত হবে।
